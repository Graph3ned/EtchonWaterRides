# How to Convert USER_MANUAL.html to PDF

## Method 1: Using Chrome/Edge (Recommended)

1. **Open the HTML file:**
   - Double-click `USER_MANUAL.html` to open it in your default browser
   - Or right-click the file → "Open with" → Choose Chrome or Edge

2. **Open Print Dialog:**
   - Press `Ctrl + P` (or `Cmd + P` on Mac)
   - Or click the menu (three dots) → "Print"

3. **Configure Print Settings:**
   - **Destination**: Select "Save as PDF" or "Microsoft Print to PDF"
   - **Layout**: Portrait
   - **Paper size**: A4
   - **Margins**: Default or Custom (adjust if needed)
   - **Options**: 
     - ✅ Check "Background graphics" to include colors
     - ✅ Check "Headers and footers" (optional)

4. **Save the PDF:**
   - Click "Save" or "Print"
   - Choose a location and filename (e.g., `USER_MANUAL.pdf`)
   - Click "Save"

## Method 2: Using Firefox

1. Open `USER_MANUAL.html` in Firefox
2. Press `Ctrl + P` (or `Cmd + P` on Mac)
3. Select "Save to PDF" as the destination
4. Click "Save"
5. Choose location and filename
6. Click "Save"

## Method 3: Using Online Converters

If you prefer an online tool:

1. Go to an online HTML to PDF converter such as:
   - https://www.ilovepdf.com/html-to-pdf
   - https://www.freepdfconvert.com/html-to-pdf
   - https://html2pdf.com/

2. Upload `USER_MANUAL.html`
3. Configure settings (A4, Portrait)
4. Convert and download

## Tips for Best Results

- **Use Chrome or Edge** for best formatting
- **Enable "Background graphics"** to preserve colors and styling
- **Check the preview** before saving to ensure everything looks correct
- The HTML file is already optimized for A4 paper size
